二、文本预处理


re 是 Python 中的一个内置模块，用于对字符串进行正则表达式匹配和操作。它提供了一组功能强大的工具，可用于处理和操作文本数据。可以使用 re 模块来执行以下操作：

  1. 正则表达式匹配：使用 re.match()、re.search() 或 re.findall() 等函数来查找符合特定模式的文本。
  2. 字符串替换：使用 re.sub() 函数来进行文本中的模式替换。
  3. 分割字符串：使用 re.split() 函数来根据模式对字符串进行分割。
  4. 其他文本操作：比如提取、匹配、搜索、替换等。
  5. 通过使用 re 模块，可以更灵活地处理和操作文本数据，实现各种复杂的模式匹配和文本处理任务。

此处，r'[^\u4e00-\u9fa5]'是一个正则表达式，它的意思是匹配任何不在\u4e00到\u9fa5范围内的字符，也就是任何非中文字符。re.sub是一个函数，它的功能是用一个新的字符串替换文本中匹配正则表达式的部分。
''是一个空字符串。
即用空字符串替换文本中的所有非中文字符，也就是删除文本中的所有非中文字符，只保留中文字符。

jieba是一个流行的中文文本处理库，它提供了中文分词的功能。它具有高性能和易用性，并且在处理中文文本时非常实用。它可以帮助你对中文文本进行分词、词性标注等处理。
